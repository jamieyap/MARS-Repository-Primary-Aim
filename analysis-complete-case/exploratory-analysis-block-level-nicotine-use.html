3 Time-varying association between engagement in self-regulatory strategies and nicotine use on current person-block

\[ \text{logit} Pr\left(\text{ANY NICOTINE USE}_{i} \left(t_{i,j} \right) = 1\right) = \boldsymbol{Z}_c\boldsymbol{\alpha} + \beta_0 \left(t_{i,j}\right) + \beta_1 \left(t_{i,j}\right) \text{c_ANY PROMPT} \left(t_{i,j}\right) + \beta_2 \left(t_{i,j}\right) \text{WHETHER ENGAGED} \left(t_{i,j}\right) \] where \(\boldsymbol{Z}_c\) denotes centered baseline covariates.

5 Time-varying association between engagement in self-regulatory strategies and nicotine use on current person-block

\[ \text{logit} Pr\left(\text{ANY NICOTINE USE}_{i} \left(t_{i,j+1} \right) = 1\right)= \boldsymbol{Z}_c\boldsymbol{\alpha} + \beta_0 \left(t_{i,j}\right) + \beta_1 \left(t_{i,j}\right) \text{c_ANY PROMPT}_{i} \left(t_{i,j} \right) + \beta_2 \left(t_{i,j}\right) \text{c_ANY NICOTINE USE}_{i} \left(t_{i,j} \right) + \beta_3 \left(t_{i,j}\right) \text{WHETHER ENGAGED} \left(t_{i,j}\right) \]

where \(\boldsymbol{Z}_c\) denotes centered baseline covariates.
